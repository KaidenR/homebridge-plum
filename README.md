# homebridge-plum
Plum plugin for [HomeBridge](https://github.com/nfarina/homebridge)

Connects homebridge to the Plum Lightpad API.

# Configuration

Configuration sample:

 ```
"platforms": [
        {
            "platform": "Plum",
                        
            "username" : "username",
            "password" : "password"
        }
    ],

```
